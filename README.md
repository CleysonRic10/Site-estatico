# Site-estatico
Linguagem HTML, CSS e JS simples.
Conteúdo fixo.
Site one-page (página única): é um site que contém todo o seu conteúdo (informações, serviços, contato, etc.) em uma única página HTML, sem a necessidade de o usuário clicar em links para acessar outras páginas; a navegação é feita através de rolagem (scroll) e links internos que levam a diferentes seções da mesma página. É uma abordagem simplificada, moderna e focada em dispositivos móveis, ideal para apresentar um negócio ou produto de forma concisa e direta. 
Responsividade para Desktop, Tablet e Mobile.
Valores depende das necessidades basicas.
